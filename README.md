# 👁️ Sistema de Gestión para Óptica

Este es un sistema web integral diseñado para automatizar los procesos de una óptica, permitiendo llevar un control organizado de pacientes y sus historias clínicas.

## ✨ Funcionalidades Principales
* *Módulo de Pacientes:* Registro completo, edición y consulta de datos personales.
* *Historias Clínicas:* Gestión digital de consultas, fórmulas y evolución del paciente.
* *Seguridad y Acceso:* Sistema de autenticación para usuarios autorizados.
* *Arquitectura:* Desarrollado bajo el patrón *MVC* (Modelo-Vista-Controlador).

## 🛠️ Tecnologías Utilizadas
* *Backend:* PHP 8+
* *Base de Datos:* MySQL (MariaDB)
* *Frontend:* HTML5, CSS3, Bootstrap 5 y JavaScript (jQuery).
* *Entorno:* XAMPP / Servidor Apache.

## 🚀 Instalación y Configuración
1. *Clonar el repositorio:* En tu carpeta htdocs, abre la terminal y usa git clone https://github.com/nymos10531/mi-primer-proyecto.git.
2. *Base de Datos:* Crea una base de datos llamada optica en phpMyAdmin e importa el archivo database/optica.sql.
3. *Conexión:* Revisa el archivo lib/conf/connection.php para asegurar que los credenciales de tu MySQL sean correctos.
4. *Ejecución:* Accede desde tu navegador a localhost/mi-primer-proyecto/web/index.php.
